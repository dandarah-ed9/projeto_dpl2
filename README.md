Requisitos

Elabore um projeto utilizando o elemento Canvas e outros componentes do HTML
que seja um jogo original.
- O jogo deve implementar o conceito de gravidade (Dica: comece por essa
implementação);
- O jogo deve ter capacidade de ser 1 player ou 2 players;
- O jogo não deve ser réplica/cópia de nenhum outro jogo existente;
- Conceitos avançados só podem ser implementados após autorização (não utilizar
IA deliberadamente).
